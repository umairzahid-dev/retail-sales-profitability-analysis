# Retail Sales & Profitability Analysis
This project analyzes retail sales data to uncover business insights related to sales performance, profitability, discount strategies, and product categories.

The analysis was performed using Python, Pandas, NumPy, Matplotlib, Seaborn, and Scikit-Learn.
## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook
## Dataset

Superstore retail dataset containing:

- Orders
- Products
- Customers
- Sales
- Profit
- Discount
- Region
- ## Analysis Performed

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Category Analysis
- Profitability Analysis
- Discount Analysis
- Sales Trend Analysis
- Correlation Analysis
- Feature Engineering
- Machine Learning Experiment
- ## Key Insights

- Technology generated the highest profit.
- Sales was the strongest contributor to profitability.
- Higher discounts negatively impacted profit.
- Tables and Bookcases generated losses despite strong sales.
- November and December recorded peak sales.
- ## Machine Learning Experiment

Random Forest and Linear Regression models were used to explore profit prediction.

The experiment highlighted the importance of feature engineering and the limitations of predicting profit without cost-related variables.
